# GoldFire framework for Meteor

## dependencies
* aldeed:collection2
* iron:router
* aldeed:autoform
* coffeescript

## install
```
meteor add spectrum:goldfire
```

GoldFire will add packages to need install automatically.

## How to use
### add a collection
(in meteor shell)
```
Meteor.call('generator.collection', 'Posts');
```
